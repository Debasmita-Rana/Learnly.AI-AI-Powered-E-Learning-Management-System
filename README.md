# Learnly.AI – AI Powered E-Learning Management System
Learnly.AI is a full-stack MERN-based AI-powered E-Learning Management System designed to provide an interactive and personalized learning experience. The platform supports role-based access for Admin, Instructor, and Students with features for course management, quizzes, assignments, AI assistance, and secure online payments.

## 🚀 Features

### 👤 Authentication & User Management
- User Registration and Login
- JWT Authentication
- Role-Based Access Control (Admin, Instructor, Student)
- User Profile Management

### 📚 Course Management
- Create, View, and Delete Courses
- Course Categories and Pricing
- Lesson Upload and Management

### 🎓 Learning System
- Course Enrollment
- My Courses Dashboard
- Learning Progress Tracking

### 📝 Quiz & Assignment Module
- Create and Manage Quizzes
- Automatic Quiz Evaluation
- Quiz Result Management
- Assignment Submission System
- View Student Submissions

### 🤖 AI Features (Groq AI)
- AI Tutor for Personalized Learning
- AI Summary Generator
- AI Quiz Generator
- Intelligent Topic Explanations and Recommendations

### 🛒 Cart & Payment System
- Add Courses to Cart
- Remove Courses from Cart
- Buy Courses using Razorpay Payment Gateway
- Order Management
